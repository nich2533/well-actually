# src/

Your application code lives here. This starter ships empty on purpose — the point of `well-actually` is the *context* layout around your code (`CLAUDE.md`, `.claude/rules/`, `documentation/`), not the code itself.

Drop your project in, then make the `.claude/rules/` and `documentation/` files describe it — and tune each rule's `paths:` glob to match where that code actually lives.
